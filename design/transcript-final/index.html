Правила группировки

Один говорящий → одна группа bubble. Пока человек говорит, новые фразы добавляются continuous-стилем (gap уменьшен, верхний corner смягчён). Tail (острый угол вниз) — только у последнего bubble группы. При смене говорящего предыдущая группа финализируется, новая открывается на противоположной стороне.

Длинные реплики

Bubble не растёт бесконечно: при 240+ символах финализируется и продолжается следующим bubble той же группы (continuous). Это сохраняет читаемость, не создаёт «простыни», и максимум 3 последних bubble видны одновременно — старые плавно уходят вверх.

Live state

Пока человек ещё говорит, в конце текущего bubble — пульсирующие точки (•••). Когда речь завершена (пауза или другой собеседник вступил) — точки исчезают, bubble переходит в финализированное состояние.

Параллельный язык

Основной текст (большой, светлый) — «мой» язык. Курсивом снизу — перевод. Для моих реплик primary = оригинал; для собеседника primary = перевод. Я всегда читаю на родном языке, фоном вижу версию на чужом.
