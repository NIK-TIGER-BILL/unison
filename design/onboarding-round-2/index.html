Иконки

  • Audio — speaker с двумя дугами звуковых волн (стандартный «динамик» SF Symbols-style), вместо переусложнённого equalizer'а из R1
  • Mic — классический микрофон с pill-капсулой, дугой и stand-line
  • Key — горизонтальный ключ с круглой головкой и зубцами — лучше читается как «ключ» чем диагональный из R1

Интерактив

  • Аудиодрайвер · клик «Установить» → spinner 1.8 сек (имитация .pkg installer'а с password-prompt) → done (зелёный, чек)
  • Микрофон · клик «Разрешить» → spinner 0.6 сек (имитация системного диалога TCC) → done
  • Ключ OpenAI · поле password-input с глазом для toggle visibility. Кнопка Сохранить активируется при непустом значении. Клик → done
  • Footer · «N / 3 готово» + большая кнопка «Готово» которая enabled только когда все три = done. Клик → анимация закрытия окна (placeholder)

Done-state

  • Фон карточки → светло-зелёный rgba(88,224,154,0.04)
  • Иконка → зелёная заливка rgba(88,224,154,0.14) с зелёной обводкой
  • Action-блок сворачивается (display: none), карточка ужимается
  • Справа от title — chevron-чек 16×16 (раньше был "Готово" текст)

Попробуй

  • Нажми «Установить» → подожди ~2с (spinner) → карточка зеленеет
  • Нажми «Разрешить» → быстрый spinner → done
  • Введи любой текст в поле ключа → кнопка «Сохранить» активируется
  • Используй 👁 чтобы показать/скрыть текст ключа
  • Когда все три done → footer-кнопка «Готово» становится активной
