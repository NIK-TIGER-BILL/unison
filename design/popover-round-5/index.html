Иконки

  • Settings — минимальный slider.horizontal.3 (две полоски с круглыми регуляторами). Меньше деталей чем gear-with-teeth, лучше работает в маленьком размере 14×14.
  • Call — классическая телефонная трубка. Сразу читается как «звонок», без двусмысленности.
  • Listen — наушники с дугой над ушами. «Слушаю» — про потребление аудио, а не про микрофон.

Интерактив

  • Mode-toggle — клик переключает Call / Listen, slide-animation на pill. Hover осветляет неактивную вкладку.
  • Lang side — клик открывает dropdown со всеми 10 языками. Chevron поворачивается на 180°, активная строка подсвечена розовым с галочкой.
  • Start button — состояния :hover / :active (scale 0.98) / :focus-visible. Клик переводит в режим Translating: цвет → красный «Stop», статус-точка → синяя пульсация, появляется таймер.
  • Gear button — hover-подсветка, нажатие даёт scale(0.94) — мини-feedback.

Состояния

  • ● зелёный — Ready (idle). ● синий пульсирует — Translating. (Жёлтый — reconnecting, красный — error — добавим когда нужно.)
  • Кнопка переходит из «Начать перевод» в «Остановить» с красным градиентом и stop-иконкой ▪
  • Таймер моноширинный 00:00 появляется под кнопкой, считает реальное время сессии.

Что ещё можно дотюнить

  • Подсветка кнопки :focus-visible сейчас базовая — можно сделать accent-ring аккуратнее
  • Анимация slide-pill между Call / Listen (сейчас вкладки просто меняют фон — можно добавить плавно скользящий highlight)
  • Когда меняешь язык — flag можно скейлить short bounce-in для тактильности
