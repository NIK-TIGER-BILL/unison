R4 · что изменилось
Иконка глаза заменена на текстовый toggle, подсказки укорочены, под полем ключа — ссылка на платформу OpenAI.

Toggle видимости ключа

  • Вместо иконки — «Показать» / «Скрыть» в виде маленькой ghost-кнопки внутри input (11px, мутед)
  • Hover осветляет, состояния явные и без двусмысленности
  • Accessibility-friendly — screen-reader'ы озвучивают action понятно

Сокращённые подсказки

  • BlackHole: «Нужен пароль системы.» — было длинно про 2ch/16ch
  • Микрофон: «macOS попросит подтверждение.» — убрано «в системном диалоге»
  • Заголовок без подзаголовка «Три шага и можно переводить.» — лишнее

Ссылка на OpenAI

  • Под input полем ключа — мини-строка «Получить ключ ↗»
  • Расположена слева, под input — не под кнопкой Сохранить (padding-right: 96px резервирует место для кнопки)
  • Ведёт на platform.openai.com/api-keys
  • Нейтральный цвет rgba(255,255,255,0.5) — как обычные hint-подписи. Hover осветляет до белого

Error-state · паттерн

  • Карточка получает coral-обводку и лёгкий розовый wash фона
  • Иконка слева становится coral fill + coral border
  • Error-row появляется под действием (или вместо): coral-фон, иконка ⓘ, текст в две строки (заголовок + детали), и контекстная кнопка действия
  • Кнопки восстановления: «Повторить» (audio), «Открыть Настройки» с external-иконкой (mic), inline-валидация (key)

Триггеры для демо

Используй эти кнопки чтобы посмотреть как выглядит каждая ошибка. «Повторить» в самой ошибке восстанавливает нормальный flow.

Error-states (без изменений из R3)

  • Coral-обводка карточки + coral-иконка + error-row с детальным сообщением и контекстной кнопкой
  • Audio: «Повторить» (запускает install заново)
  • Mic: «Открыть Настройки ↗» (deep-link на System Settings)
  • Key: inline-валидация — нужно sk- + 20+ символов

Попробуй

  • Введи в поле ключа abc → нажми «Сохранить» — validation error
  • Введи sk-proj-... подлиннее → save → done
  • Кликни «Показать» — пароль становится виден, label меняется на «Скрыть»
  • Кликни «Получить ключ ↗» — откроется platform.openai.com
