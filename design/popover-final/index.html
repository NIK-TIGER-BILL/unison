Что внутри
Final-итерация popover'а. Дальше идём в onboarding — все решения этого экрана зафиксированы в docs/design/DESIGN.md и переносятся.

Эстетика

  • Aurora background — radial-gradients cyan/magenta/purple на тёмном #0a0820
  • Liquid Glass material — backdrop blur 20px + saturate 160% + SVG feDisplacementMap для рефракции
  • Conic rim — переливающийся периметр по border'у через mask-composite
  • Specular highlight — top 60% area с linear+radial gradient на screen-mix

UX

  • Mode-toggle Call (телефонная трубка) ↔ Listen (наушники)
  • «Я говорю» / «Слушаю» — мини-лейблы caps читают как фраза
  • ↔ двунаправленная stroke 1.2px — нейтральная, не отвлекает
  • Lang sides кликабельны → portal-dropdown с поиском
  • Search S1 — ghost underline, без фона, на focus underline розовеет
  • Валидация — одинаковые языки → жёлтый dot + warn-row + Start disabled
  • Start states: Ready (бел градиент) / Translating (красный, timer mm:ss) / Disabled

Клавиатура

  • ↑/↓ в поиске — навигация по списку
  • Enter — выбрать выделенный
  • Esc — закрыть dropdown

Попробуй

  • Клик «🇷🇺 Русский» → откроется dropdown, autofocus в поиск
  • Набери en или 中 — фильтр
  • Выбери один и тот же язык в обе стороны — увидишь warn + дизейбл Start
  • Клик Start (когда валидно) — перейдёт в Stop + красный + timer
