Bubbles стали по-настоящему стеклянными: blur 24px + saturate 180% + SVG-displacement + specular highlight. Зона ~520px по центру.

Демо

Дизайн

  • Liquid glass восстановлен — низкая opacity background + blur(24px) saturate(180%) + SVG displacement + specular highlight ::before. Сквозь bubbles видно desktop
  • Bottom-center — зона 520px, по центру экрана, отступ 22px от низа
  • My слева, Peer справа — соответствует spatial audio (моя речь → outgoing, peer → incoming)
  • Max-width bubble 72% — не растягиваются на всю ширину зоны
  • Анимации — entrance 0.4s spring + blur(4px)→blur(0), exit 0.7s (медленнее для плавности)

Поведение «3 + 1»

  • На экране одновременно: 3 фразы фоном + 1 fade-out
  • Когда добавляется 4-я фраза, самая старая стартует fade-out (0.7s)
  • Через 700ms старая полностью убирается из DOM
  • В итоге в DOM ~4 элемента, в layout видны 3

On-screen toggle

  • Внизу под bubbles — pill с pulse-dot + timer + 👁 toggle + ⊗ close
  • Клик 👁 — скрывает только bubbles (pill остаётся, иконка меняется на eye-off, label на «Показать транскрипт»)
  • Клик ⊗ — останавливает сессию полностью (всё закрывается)
  • Pill можно перетаскивать — drag handle везде кроме самих кнопок

Попробуй

  • «+ Моя фраза» / «+ Фраза собеседника» — добавь несколько, понаблюдай fade-out на 4-й
  • «Авто-диалог» — имитация живого общения, видно как fade plays smoothly
  • 👁 в pill — скрой транскрипт, потом покажи снова
  • Перетащи pill в другое место экрана
